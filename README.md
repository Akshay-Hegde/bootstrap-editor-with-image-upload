# Image Upload feature integrated with Bootstrap WYSIWYG Editor

- Go to `js/jquery.pagedown-bootstrap.combined.js`, Line : `2370`. 
- Change your URL where you want to upload your image.

```javascript
type:'POST',
url: "upload.php", // change URL here
dataType:'json',
```
